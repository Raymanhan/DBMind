# DBMind v0.2.3

## 本次更新

- 全局国际化修复：设置页、侧栏、连接弹窗、AI 助手、会话列表、查询结果、历史记录、批量编辑、单元格编辑、SQL 确认弹窗、表设计器和 SQL 编辑器菜单均接入语言切换。
- AI 助手语言联动：SQL 生成与 SQL 优化请求会携带当前界面语言，AI 返回的解释、优化建议会按用户选择的语言输出。
- AI 默认会话同步：切换语言后，默认「新对话」标题和欢迎语会即时切换，不再残留旧语言。
- 设置界面布局修复：补齐语言设置表单样式，并优化窄屏下的设置页布局。
- 浏览器预览优化：预览模式下的关键提示会按当前语言显示。

## 安装包

- macOS Apple Silicon: `DBMind-0.2.3-mac-arm64.dmg`
- macOS Intel: `DBMind-0.2.3-mac-x64.dmg`
- Windows: `DBMind Setup 0.2.3.exe`（等待 GitHub Actions 恢复后补传）
- Linux: `DBMind-0.2.3.AppImage`（等待 GitHub Actions 恢复后补传）

## 验证

- `npm run typecheck`
- `npm run build`
- 浏览器验证语言切换后主界面、设置页、连接弹窗和 AI 助手均更新为目标语言。
