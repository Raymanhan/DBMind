# DBMind v0.2.3

## Changelog / 更新内容

### i18n — Complete internationalization / 全面国际化

- 8 languages fully translated across all UI components:
  简体中文 / 繁體中文 / English / Русский / 日本語 / 한국어 / Français / Deutsch
- Settings page: language selector with real-time switching
- Browser preview (npm run dev) bilingual support for all mock data and messages
- All error messages, tooltips, and labels translated

### Improvements / 优化

- CI release notes now auto-loaded from docs/releases/vx.y.z.md
- Release workflow requires explicit release notes file
- SOURCE_REPO_TOKEN support for private repo checkout
